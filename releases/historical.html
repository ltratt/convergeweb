Pre 1.0

Converge version numbers are of the form <major>.<minor>. Versions 0.1 to 0.8 inclusive use the old Converge base, and programs written for these versions require some small modifications before working on the new versions of Converge available in snapshots.

Version 0.8 (2006-04-04)

Downloads and instructions for the following platforms are available:
  • Unix. FreeBSD, Linux, OpenBSD on x86 are known to be supported. Other 32-bit little-endian UNIX operating systems are likely to work with little or no effort.
  • Windows [Cygwin]. This needs an up to date version of Cygwin in order to function correctly.
