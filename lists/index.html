Mailing lists

All mailing lists are run using the mlmmj mailing list manager. All interactions with the mailing list manager are via e-mail. Commands are of the form list+command@convergepl.org where the list name and the command are separated by a plus '+' character. e.g. to subscribe to the announce list, send an e-mail to announce+subscribe@convergepl.org.

The following e-mail lists are dedicated to Converge. For convenience you can subscribe and unsubscribe via this web-page (you will get confirmation e-mails to prevent miscreants abusing this system):

announce@convergepl.org Low-traffic announcements [moderated]. Local archive.
